Overview

Horizon Insurance is a professional website created for an independent insurance advisor in Canada. The goal of the project was to build a modern and trustworthy online presence where users can explore coverage options, understand services quickly, and take action through a simple quote request process. The website combines informative content with conversion-focused design to support both user trust and business goals.

Problem

Insurance websites often feel complicated, text-heavy, and difficult for users who are simply trying to understand their options. This project needed to present multiple insurance categories in a clear way, reduce confusion around premium estimation, and make it easier for visitors to contact an advisor without feeling overwhelmed by technical language.

Solution

I designed and developed a clean multi-section website with strong visual hierarchy, supportive content, and clear calls to action. The experience guides users through services, a premium calculator, trust indicators, FAQs, and a contact form. This structure helps visitors learn about coverage, interact with the calculator for a quick estimate, and confidently request a quote.

Key Features

  • Hero section with clear messaging and strong call-to-action buttons
  • Service cards for Life, Health, Critical Illness, and Disability Insurance
  • Interactive insurance premium calculator for quick monthly estimate guidance
  • Trust-building content including testimonials, business metrics, and advisor positioning
  • FAQ section to answer common user questions before inquiry
  • Lead generation contact form for quote requests and follow-up
  • Responsive layout optimized for desktop and mobile browsing

Design Approach

The design direction focused on clarity, professionalism, and accessibility. I used generous spacing, readable typography, strong section separation, and a consistent blue visual language to communicate trust and stability. The layout was planned to make important information easy to scan while keeping the overall experience polished and user-friendly.

Development

This project was developed as a responsive front-end website with structured sections, reusable styling, and interactive functionality for the premium calculator and contact workflow. The build supports a simple and practical user journey, making it suitable as both a portfolio project and a real-world business website concept.

Outcome

The final website presents insurance information in a more approachable way and gives users multiple entry points to engage with the business. By combining service education, quick estimation, and a clear inquiry path, the project demonstrates how thoughtful web design can improve trust, communication, and conversion for service-based businesses.

Technologies

The project used front-end tools and responsive design practices to create a smooth and modern web experience.
