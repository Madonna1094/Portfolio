PORTFOLIO WEBSITE FILE GUIDE

1. Open index.html in your browser.
2. Replace these image files with your own images:
   - assets/profile-photo.jpg
   - assets/kfc-thumbnail.jpg
   - assets/rose-stone-thumbnail.jpg
   - assets/splush-thumbnail.jpg
   - assets/budgetbites-thumbnail.jpg
   - assets/upskill-thumbnail.jpg
   - assets/graphic-placeholder.jpg

3. Add your real links in the project pages:
   - View Case Study -> your Behance project link
   - View Website -> your live website link

4. Edit text inside:
   - about.html
   - contact.html
   - project pages

5. To add a new project:
   - Duplicate any project page
   - Update the text, image, and links
   - Add the new card in projects.html

6. Main categories already included:
   - Graphic Design
   - UI/UX Design
   - Web Design & Development
